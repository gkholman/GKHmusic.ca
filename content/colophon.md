<h2 style="font-size:1.6rem;color:var(--brown-dark);margin-bottom:1rem;">Provenance</h2>
<p style="color:var(--text-mid);margin-bottom:1.25rem;">
  This Electronic Press Kit was conceived, designed, and is maintained by G. Ken Holman
  in collaboration with Anthropic Claude. The division of labour is straightforward:
  Ken is the Webmaster; Claude is the Webslave.
</p>
<p style="color:var(--text-mid);margin-bottom:2.5rem;">
  All content — photographs, biographical text, performance descriptions, and editorial
  decisions — originates with Ken. Claude handles structural design, code generation,
  and site maintenance on instruction.
</p>

<h2 style="font-size:1.6rem;color:var(--brown-dark);margin-bottom:1rem;">Technology</h2>
<p style="color:var(--text-mid);margin-bottom:1.25rem;">
  The site is built with <a href="https://gohugo.io" target="_blank" rel="noopener">Hugo</a>,
  a static site generator. Source files are maintained in a private GitHub repository
  under version control. Deployment is automated via GitHub Actions, which builds the
  Hugo site and pushes the generated output to a virtual private server hosted at
  <a href="https://CraneSoftwrights.com" target="_blank" rel="noopener">Crane Softwrights Ltd.</a>
</p>
<p style="color:var(--text-mid);margin-bottom:1.25rem;">
  The domain <strong>GKHmusic.ca</strong> is a Canadian registration served from
  the same VPS infrastructure as CraneSoftwrights.com, managed through WHM and cPanel.
</p>
<p style="color:var(--text-mid);margin-bottom:2.5rem;">
  Lightbox functionality is provided by
  <a href="https://biati-digital.github.io/glightbox/" target="_blank" rel="noopener">GLightbox</a>.
  Typography is set in Cormorant Garamond and Source Serif 4, served via Google Fonts.
  The section dividers reproduce a photograph of the Longido-sourced guitar strap
  belonging to Nashipai, Ken's custom 12-string guitar.
</p>

<h2 style="font-size:1.6rem;color:var(--brown-dark);margin-bottom:1rem;">Design</h2>
<p style="color:var(--text-mid);margin-bottom:1.25rem;">
  The colour palette is drawn from two sources: the earthy brown tones of Ken's
  business card, and the deep navy of the Maasai-patterned drapery used as a
  backdrop for photographs of Nashipai. The decorative motif — geometric symbols
  in the mud cloth tradition — is taken directly from the guitar strap sourced
  in Longido, Tanzania.
</p>
<p style="color:var(--text-mid);margin-bottom:2.5rem;">
  The design intent is understated and honest: a performing musician's professional
  presence, not a marketing exercise.
</p>
